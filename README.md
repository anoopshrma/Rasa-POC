# Rasa-POC
RASA POC PROJECT 

## Requirements 
- Python version : 3.8
- Create Python Environment and activate it.


### Installation cmd
``` 
pip install -r requirements.txt 
```

### Run Rasa and Action Server
```
rasa run -m models --enable-api --cors "*"

rasa run actions
```

## To run the widget open the index.html file in your browser
